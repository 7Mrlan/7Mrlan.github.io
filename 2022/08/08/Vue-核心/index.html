Vue-核心

发表于 2022-08-08 分类于 Vue

官方解释

Vue (读音 /vjuː/，类似于 view) 是一套用于构建用户界面的渐进式框架。与其它大型框架不同的是，Vue 被设计为可以自底向上逐层应用。Vue 的核心库只关注视图层，不仅易于上手，还便于与第三方库或既有项目整合。另一方面，当与现代化的工具链以及各种支持类库结合使用时，Vue 也完全能够为复杂的单页应用提供驱动
source:https://cn.vuejs.org/v2/guide/

以上是官方对vue的描述，这里提到的渐进式框架，我举个例子帮助理解什么是渐进式，如果你的公司以往都是使用Jquery框架，现在想更换vue框架，你可以逐步的删除关于Jquery的代码，然后逐步用Vue进行项目重构。当然这并没有那么重要。我觉得比较重要的有三个方面

  • 响应式
  • 声明式编程
  • MVVM框架

我会逐步对他们进行展开讨论的，你只需知道现在它们会是我写博客的重点部分，我们暂时不讨论它们

vue的引入

先来建一个vue实例,你可以使用三种方式引入vue框架

  1. 下载vue.js文件

https://cn.vuejs.org/v2/guide/installation.html

  1. 利用CDN引入（内容分发网络）

<script src=”https://cdn.jsdelivr.net/npm/vue@2.7.8/dist/vue.js"></script>,我这里是开发环境版本，不太推荐生产版本。

  1. npm

$ npm install vue

第一个vue Hello World

上面这些可以自行去官网上查找，我们直接进入正题

在我们引入的vue.js中，已经为我们定义好了Vue类，对类不明白的可以去看我的class类的介绍，我们有了Vue这个类作为模板，去创建实例化的对象，我们将对象作为参数传到构造器里。这里我们传了两个属性

  1. el代表元素element的缩写，它的作用是将vue实例挂载到DOM对象上，一旦挂载了vue实例，这里DOM就能被vue进行管理解析，vue即会进行数据绑定，当数据发生变化的时候它会响应式的更新DOM，同时也会对DOM进行监听，当DOM的事件的触发使数据发生改变时，vue也会实时的去更新data里面的数据，这就是MVVM模型
  2. data代表数据，它可以由原生js传入静态对象作为数据，也可以传入向服务器请求的数据它属于MVVM模型中的model

利用这两个属性，vue就可以对它们同时进行管理，使得数据能响应式的更新，你可以试着在控制台输入。app.message = ‘鸡你太美’，我们这个时候可以发现DOM的内容同步的更新了。
